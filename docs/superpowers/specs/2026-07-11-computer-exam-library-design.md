# 机考题库与题库分类页重构设计

## 目标

在顶部导航新增独立“机考题库”栏目，从 `可视化积分区域题目汇总.md` 导入无需用户补充参数、可直接生成积分区域的题目。重构往年真题与机考题页面，使两类题库共享题目展示能力，同时分别保留按学年和按专题浏览的最佳方式。

## 信息架构

- 顶部导航为“在线计算 / 往年真题 / 机考题库”，默认仍进入在线计算。
- 往年真题路由保持 `/exams`，按学年、积分类型和关键词筛选。
- 机考题库使用 `/computer-exams`，按源文件中的专题标题分类。
- 专题名称完整保留，例如 `8.1.1 二重积分的概念及性质`、`9.8.1 斯托克斯公式`。
- 桌面端使用专题侧栏，移动端使用专题下拉选择，均显示各专题实际导入题数。

- 每题保存稳定 ID、专题代码与名称、题号、页码、积分类型、题干、区域、答案、解析和 `IntegralSpec`。
- 生成结果保存元数据：源题总数、成功导入数、各专题数量、排除数与排除原因。

## 可视化筛选规则

题目只有同时满足以下条件才进入网站：

1. 区域字段完整且能解析出等式或不等式；
2. 不含需要用户赋值的自由参数，如 `a`、`R`、`h`、`t`；`x/y/z`、数值常量和 `\pi` 不视为自由参数；
3. 能安全映射为二重积分、三重积分、隐式曲线或隐式曲面的条件区域；
4. 扫描范围可由常数边界或几何条件可靠推导；
5. 生成的 `IntegralSpec` 能通过绘图构建验证。

任何一步失败都排除该题并记录原因。网站不显示不可直接可视化的题，也不要求用户补充参数。

## 页面重构

- 抽取共享题目卡片，统一渲染富文本题干、元信息、答案/解析折叠区和“一键在计算台可视化”。
- 往年真题卡片显示学年、来源、分值与积分类型。
- 机考题卡片显示专题、题号、页码与积分类型。
- 搜索覆盖题干、区域、专题和解析。
- 机考题专题侧栏支持“全部专题”和单专题选择，并显示结果数量。
- 空搜索结果、空专题和解析缺失均使用明确提示，不显示空白页面。

## 一键可视化

- 点击机考题的按钮，通过路由状态把预生成 `IntegralSpec` 载入在线计算台。
- 二重、三重题优先使用条件区域模式。
- 曲线、曲面题优先使用隐式条件模式。
- 计算台仍负责实际 Plotly 可视化、视角旋转和后续计算能力。
- 导入前的验证保证题库中每个可视化按钮都有可用区域，不出现占位按钮。

## 性能

- Markdown 解析只在生成阶段执行，不进入浏览器包。
- 机考题页面使用懒加载。
- 搜索索引在模块加载时构建，输入使用 deferred value，避免 600 题规模下频繁阻塞。
- 专题筛选先缩小集合，再执行渲染；题目列表沿用轻量卡片结构。

## 测试与验收

- 解析测试覆盖专题切换、题号、页码、多行 LaTeX、答案和解析。
- 筛选测试覆盖自由参数、缺失区域、无法推导扫描范围和支持的数值区域。
- 每个导入题运行 `buildPlotSpec` 验证；失败题不能写入最终数据集。
- 浏览器验证三个导航入口、专题侧栏、移动端下拉、搜索、答案折叠和一键可视化。
- 回归验证往年真题原有学年/类型筛选和计算台载入不受影响。

## 非目标

- 不导入需要用户给自由参数赋值的题。
- 不提供做题、判分、错题本或收藏功能。
- 不在浏览器中动态读取源 Markdown。
- 不修改题目原答案和解析内容。
