# 隐式曲线与曲面条件区域实施计划

1. 扩展 `LineIntegralSpec`、`SurfaceIntegralSpec`，加入参数/条件模式和共享 `ConstraintRegion`，保持旧题库默认参数模式。
2. 从 `BoundsEditor` 抽取共享条件编辑器，支持空白初始状态、扫描范围、增删约束和显式转换按钮。
3. 二重/三重积分切换条件模式时不再自动填充；主动转换时才从上下界生成约束；三维体区域移除内部散点。
4. 曲线积分与曲面积分加入模式切换。参数界面保持原状，条件界面使用 LaTeX 约束和三轴扫描范围。
5. 在 `plotSpec` 中增加隐式曲面等值面渲染，以及两个等式交集的隐式曲线采样渲染；附加不等式负责截取。
6. 在计算入口拦截无法可靠参数化的隐式曲线/曲面，显示明确说明，不启动 Python。
7. 添加数据、转换、绘图和计算保护测试；运行全部测试与生产构建。
8. 使用浏览器验证桌面和移动端的切换、编辑、绘图、旋转、错误提示及题库回归，最后提交功能修改。
