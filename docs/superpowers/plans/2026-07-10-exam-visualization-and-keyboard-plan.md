# 真题一键可视化与分组键盘实施计划

## 1. 建立严格题库生成层

- 增加真题公式规范化与显式累次积分解析器。
- 增加人工校正规格表，覆盖题意明确但无法从统一格式直接解析的区域。
- 为每个候选规格运行字段、表达式和绘图采样验证。
- 生成只包含通过验证题目的网页题库，并记录排除原因供维护。

## 2. 接通真题到计算台的数据流

- 扩展题库类型，为保留题目附加完整 `IntegralSpec`。
- 真题页仅加载严格题库并更新统计。
- 把题目按钮改为“一键在计算台可视化”，通过路由状态传递规格。
- 计算台安全接收并持久化传入规格，立即绘制对应区域。

## 3. 扩展数学符号键盘

- 把符号定义拆到独立模块并分成常用、积分、函数、字母、向量五组。
- 实现可访问的分类切换和 MathLive 占位符插入。
- 调整桌面和手机端样式，保持计算台首屏密度。

## 4. 测试与视觉验收

- 增加题库生成、全部规格绘图、路由载入和键盘定义测试。
- 运行测试与生产构建。
- 用内置浏览器验证真题筛选、一键载入、五类积分和运算符分类。
- 用桌面及手机视口截图对照既有设计，清理临时 QA 产物后提交。
