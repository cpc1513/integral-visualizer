# 机考题库实施计划

1. 编写 Markdown 生成脚本，解析专题、题号、页码、题干、区域、答案和解析。
2. 对区域执行自由参数检查、约束标准化、扫描范围推导和积分类型映射，生成静态 JSON 与排除统计。
3. 为生成数据增加结构测试和逐题 `buildPlotSpec` 验证，失败题不得进入最终题库。
4. 抽取共享题目卡片、搜索文本和类型标签，重构往年真题页面复用它们。
5. 新增机考题库页面、专题侧栏、移动端专题下拉、搜索和结果计数。
6. 增加 `/computer-exams` 懒加载路由与顶部导航入口，保持在线计算为默认页。
7. 验证机考题一键载入计算台、往年真题筛选回归、桌面/移动端布局和控制台状态。
8. 运行测试、生产构建、差异检查并提交。
